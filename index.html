本アンケート調査は、陸連公認マラソン大会における心肺停止事例を収集し、リスクの高い大会の特徴を明らかにすることを目的としています。

+

以下の説明をご確認のうえ、ご回答をお願いいたします。

回答の流れ

+

1⃣大会参加者数について

+

参加者数はスタートラインを超えたランナーの数で、申し込み人数（エントリー数）ではありません。

+
    +
  • 男性（50歳超男性割合、60歳超男性割合含む）の割合は100％単位でご記入ください（小数点第一位）。
    • +
  • 割合は参加者数に対する割合ですが、完走者数でも代用可能です（例：男性完走者数／完走者数）。
    • +
  • すでに入力済みのデータは過去にご報告いただいたデータ、ないしrunnet.jpの記録から取得しています。
    • +
  • 入力済みデータが間違っている場合は、修正ボタンを押して数値を上書き下さい。
    • +
+ +

2⃣心肺停止の定義

+

レース中またはフィニッシュ直後に急に倒れ、意識消失を伴い、①胸骨圧迫（心臓マッサージ）を要した、または②AEDによる解析（装着・作動を含む）が行われた症例とします。

+
    +
  • 未報告のレースについてご回答ください（すでに回答済みレースで間違いがある場合も修正可能です）。
    • +
  • 胸骨圧迫を行わず自然に回復した症例は含みません。
    • +
  • AED装着前に胸骨圧迫のみで回復した症例は含みます。
    • +
  • 判断に迷う症例は疑いも含めてご回答ください。
    • +
  • 本フォームの回答内容は、研究担当者以外が個別に閲覧することはありません。
    • +
  • アンケート結果をもとに後日確認のメールをする場合があります。
    • +
